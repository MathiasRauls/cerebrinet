# Cerebrinet

## Mission
We aim to build the next stage in global communication far removed from the centralized cellular technocratic oligarchs of today. Freeing us all from the physical limitations of our "free speech". We're going to build a non invasive BCI model that decodes EEG patterns associated with imagined or intended speech into structured linguistic representations. 

**TLDR**: I am attempting to build a BCI model for "telepathy" because I hate miscommunication, & typing is too damn slow.
